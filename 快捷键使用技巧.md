# 快捷键备忘（Trae / VSCode）

---

## 光标与行操作

| 操作 | Windows/Linux | macOS |
| :--- | :--- | :--- |
| 在当前行**下方**插入新行 | `Ctrl + Enter` | `Cmd + Enter` |
| 在当前行**上方**插入新行 | `Ctrl + Shift + Enter` | `Cmd + Shift + Enter` |
| 切换自动换行（显示方式） | `Alt + Z` | `Option + Z` |
| 光标移动到上一行（Trae 需手动绑定） | `Ctrl + ↑` | `Ctrl + ↑` |
| 光标移动到**行首第一个字母** | `Home` | `Cmd + ←` |
| 跳转到指定行号 | `Ctrl + G` | `Cmd + G` |
